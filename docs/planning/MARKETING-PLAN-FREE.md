# Qepton Free - Marketing Plan

> **Issue Tracking:** See [GITHUB-ISSUES-PREVIEW.md](./GITHUB-ISSUES-PREVIEW.md) for full issue details.

## Executive Summary

Qepton Free is a code snippet manager powered by GitHub Gist, offering developers a streamlined way to organize, search, and access their code snippets across platforms. This marketing plan outlines strategies to grow the user base and establish Qepton as the go-to gist management tool.

## Target Audience

### Primary

- **Individual developers** who use GitHub Gists for code snippets
- **Students and bootcamp learners** building their code libraries
- **Open source contributors** who share code frequently

### Secondary

- **Technical writers** who maintain code examples
- **Educators** who share code with students
- **DevRel professionals** who create and share demos

## Value Proposition

**"Your gists, organized and accessible everywhere."**

Key differentiators:
- Native desktop app (Electron) + PWA + mobile (Capacitor)
- Fast fuzzy search across all gists
- Automatic language detection and tagging
- Custom tagging system for organization
- Markdown and Jupyter notebook rendering
- Offline access to cached gists

## Messaging Guidelines

### Tone

- Developer-friendly and technical
- Humble and helpful, not salesy
- Focus on solving real problems

### Key Messages

1. "Stop losing your best code snippets"
2. "GitHub Gists, supercharged"
3. "Search 1000 gists in milliseconds"
4. "Your snippets, everywhere you code"

### Avoid

- Overpromising features
- Criticizing competitors directly
- Non-developer marketing speak
