# Qepton Demo

**Try Qepton without creating an account!**

## [Launch Demo](https://qepton-demo.github.io)

**URL:** https://qepton-demo.github.io

This is an isolated playground for exploring Qepton's features. Complete a simple CAPTCHA to access a shared demo environment with sample code snippets.

### Features to Try

- Browse and search code snippets
- View syntax-highlighted code in 20+ languages
- Render Jupyter notebooks and Markdown
- Organize with tags and favorites
- Test the editor (changes reset weekly)

### Limitations

- Shared account (others may see your changes)
- Data resets every Sunday at midnight UTC
- Some features disabled for security

### Ready for More?

Get your own private snippet library:

1. Visit [Qepton](https://whizbangdevelopers-org.github.io/Qepton/)
2. Enter your GitHub Personal Access Token
3. Your gists become your snippet library
